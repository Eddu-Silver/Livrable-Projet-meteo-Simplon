# Application Météo pour Transports en Commun

Après quelques heures de travail acharné et d'apprentissages nouveaux, j'ai le plaisir de vous envoyer, enfin, l'exercice livrable demandé.  
Les modifs permettent à l'appli d'afficher la météo actuelle d'une ville pré-configurée, Nantes en l'occurrence, avec rafraîchissement automatique toutes les heures.

Pour faire cet exercice, j'ai commencé par demander quelques conseils de bases à une amie puis, je me suis lancé à l'aide d'une IA à qui j'ai demandé de m'accompagner à la manière d'un professeur.  
J'ai laissé dans les codes, plusieurs commentaires afin de pouvoir y revenir si nécessaire lors de mon apprentissage.

## 🚀 Fonctionnalités

- ✅ Affichage de la météo actuelle (température, humidité, pression, vent)
- ✅ Lever et coucher du soleil
- ✅ Températures minimales et maximales du jour
- ✅ Icônes et descriptions météo
- ✅ **Ville configurable** via fichier `config.json`
- ✅ **Rafraîchissement automatique** toutes les heures
- ✅ Utilisation de l'API **Open-Meteo** (gratuite, sans clé)

## 🛠️ Technologies utilisées

- [Next.js](https://nextjs.org/) - Framework React
- [Open-Meteo API](https://open-meteo.com/) - Données météo
- CSS Modules - Styles
